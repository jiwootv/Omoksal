## 기본적인 알고리즘
각각의 돌 1칸 이내에 다른 돌이 위치하지 않은 곳을 각각 지정하고, 그곳에 돌을 깔았을 때 상황에 따른 점수를 부여하는 방식입니다. 그걸로 점수가 가장 높은 것을 찾고 그것들 중 중앙에 가까운것만 추려냅니다. 그래도 여러개면 그냥 하나 찍습니다.
